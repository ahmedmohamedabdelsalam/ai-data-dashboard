# AI Data Analysis Dashboard - Backend API

FastAPI backend for the AI Data Analysis Dashboard. Provides endpoints for data analysis, anomaly detection, correlation analysis, and AI-powered insights.

## API Endpoints

- `GET /api/health` - Health check
- `POST /api/upload` - Upload a CSV file for analysis
- `POST /api/full_analysis` - Perform full data analysis
