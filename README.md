# AI Data Analysis Dashboard

A comprehensive, production-ready platform for automated data analysis and artificial intelligence-driven insights. This project streamlines the process of dataset exploration, anomaly detection, and natural language report generation using a FastAPI backend and a Next.js frontend.

## Key Features

- Multiformat Upload: Support for CSV and Excel (XLSX) datasets.
- Automated EDA: Generation of descriptive statistics and data health summaries.
- Advanced Analytics: Correlation analysis, distribution vectors, and feature grouping.
- Intelligent Anomaly Detection: Outlier identification using the Isolation Forest algorithm.
- AI Executive Briefing: High-level narrative insights generated via OpenRouter (Mistral/OpenAI models).
- Production Infrastructure: Modular service architecture, environment-based configuration, and structured logging.

## Project Architecture

The system is split into two primary components:

- **Backend (FastAPI)**: Handles compute-heavy data processing, ML execution, and LLM integrations.
- **Frontend (Next.js)**: Provides an interactive dashboard for data visualization and orchestration.

```
.
├── backend/                # FastAPI Application
│   ├── app/
│   │   ├── api/            # Route definitions
│   │   ├── services/       # Core analysis logic (Modular Service Layer)
│   │   ├── models/         # Pydantic schema definitions
│   │   └── config/         # System settings and validation
│   └── requirements.txt    # Python dependencies
└── frontend/               # Next.js Application
    ├── app/                # Pages and layouts
    ├── components/         # Reusable UI widgets
    └── services/           # API client layer
```

## Getting Started

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure environment variables (create a `.env` file):
   ```env
   OPENROUTER_API_KEY=your_api_key_here
   LLM_PROVIDER=openrouter
   LLM_MODEL=mistralai/mixtral-8x7b-instruct
   ```

4. Start the server:
   ```bash
   uvicorn app.main:app --reload
   ```

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## API Documentation

Once the backend is running, you can access the interactive Swagger documentation at:
`http://localhost:8000/docs`

### Primary Endpoints

- `POST /api/upload_dataset`: Upload and validate source data.
- `GET /api/full_analysis`: Orchestrate a comprehensive suite of analysis services.
- `GET /api/health`: Monitor system status.

## Analysis Methodology

- **Clean & Normalize**: Incoming data is dynamically cleaned of whitespace and normalized for numeric processing.
- **Anomaly Detection**: Uses Scikit-learn's Isolation Forest to identify global outliers based on contamination rates.
- **AI Insights**: Contextual summaries are generated by passing statistical metadata and correlation layers to advanced language models, ensuring privacy by excluding raw PII.
